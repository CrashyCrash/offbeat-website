How Harmonic Mixing Works

The Camelot Wheel assigns each key a number (1–12) and letter (A=minor, B=major). Compatible transitions are: same position (key match), ±1 number (adjacent key), and same number different letter (relative major/minor). Energy boosts go +1 (e.g. 8A → 9A).
