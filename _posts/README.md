# _posts/ — Blog Content

Drop DatBotty-generated articles here as `YYYY-MM-DD-slug.md` files.

## Front matter required

```yaml
---
layout: post
title: "Best DJ Controllers Under $300 in 2026"
date: 2026-05-11
author: Jake Morrison
description: "Top picks tested in real conditions — Pioneer, Numark, Denon."
---
```

## Naming convention

```
2026-05-11-best-dj-controllers-under-300.md
YYYY-MM-DD-slug-with-hyphens.md
```

## Automation

`hub/scripts/publish_to_html.py` can target _posts/ for long-form articles.
Run `git push` → GitHub Pages rebuilds automatically within 3–5 minutes.
